# Crop Yield Prediction using Machine Learning

##  Project Overview

This project predicts crop yield based on environmental and agricultural factors such as rainfall, temperature, pesticide usage, area, and crop type.
It uses multiple Machine Learning techniques including Regression, Classification, and Clustering to analyze and compare performance.

---

##  Objectives

* Analyze agricultural dataset
* Predict crop yield using regression models
* Classify yield into categories (Low, Medium, High)
* Perform clustering to identify patterns
* Compare performance of different algorithms

---

##  Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

##  Features Used

* Rainfall
* Temperature
* Pesticides
* Area
* Crop

---

##  Machine Learning Models

### 🔹 Regression

* Linear Regression
* Random Forest Regressor

### 🔹 Classification

* Decision Tree Classifier
* K-Nearest Neighbors (KNN)

### 🔹 Clustering

* K-Means Clustering

---

##  Evaluation Metrics

* Mean Squared Error (MSE) – Regression
* Accuracy – Classification
* Silhouette Score – Clustering

---

##  Results

* Random Forest performed better than Linear Regression (lower MSE)
* Classification models compared using accuracy
* K-Means clustering grouped similar agricultural patterns
